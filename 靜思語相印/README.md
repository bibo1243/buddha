# 🪷 靜思語相印

將智慧話語與美好回憶融為一體的 AI 照片生成器，支援 Canon 4x6 相印機列印。

## ✨ 功能特色

### 📜 步驟一：選擇靜思語
- **隨機生成**：從精選靜思語資料庫中隨機取得
- **自訂輸入**：可輸入自己喜歡的語句
- **分類篩選**：依慈悲、智慧、感恩、精進、安心等分類選擇

### 📷 步驟二：上傳照片
- 支援拖曳或點擊上傳
- 支援 JPG、PNG 格式
- 即時預覽功能

### 🎨 步驟三：AI 生成
- **3D Q版柔和**：可愛的 3D 卡通風格
- **水彩風格**：柔美的水彩畫風
- **禪意水墨**：傳統中式水墨風格

### 📤 輸出功能
- 4" × 6" 橫式相片 (1800 × 1200 px @ 300 DPI)
- 適用於 Canon SELPHY 等相片印表機

## 🤖 Google AI Imagen 3 整合

本程式支援 **Google AI Imagen 3** 直接生成高品質圖片！

### 設定方式

1. 點擊右上角的 ⚙️ **設定按鈕**
2. 輸入您的 **Google AI API Key**
   - 您可以到 [Google AI Studio](https://aistudio.google.com/apikey) 免費取得 API Key
3. 選擇生成模式：
   - **Imagen 3 API**：使用 Google AI 直接生成精美圖片（每張約 $0.03 USD）
   - **本地合成**：生成提示詞 + 簡易預覽圖（免費）
4. 點擊「測試連線」確認 API 可用
5. 點擊「儲存設定」

### API 價格參考
- Imagen 3：$0.03 USD / 張
- Imagen 3 Fast：$0.025 USD / 張

### 支援的模型
- `imagen-3.0-generate-002`（最新版本）
- `imagen-3.0-generate-001`
- `imagen-3.0-fast-generate-001`

## 💡 兩種生成模式說明

### 模式一：Imagen 3 API（推薦）

需要 Google AI API Key，優點：
- ✅ 高品質 AI 生成圖片
- ✅ 根據靜思語意境自動生成場景
- ✅ 支援多種風格
- ✅ 直接輸出可列印的高解析度圖片

### 模式二：本地合成

不需要 API Key，優點：
- ✅ 完全免費
- ✅ 即時生成預覽圖
- ✅ 提供完整的 AI 提示詞
- ✅ 可複製提示詞到其他 AI 工具使用（Midjourney、DALL-E 等）

## 📖 靜思語資料庫

收錄證嚴法師靜思語，涵蓋以下主題：
- **慈悲**：心美、原諒、愛
- **智慧**：人生、話語、計較
- **感恩**：父母恩、知恩報恩
- **精進**：行善、時間、成就
- **安心**：平和、懺悔、淨土

## 🔒 隱私說明

- API Key 僅儲存在您的瀏覽器本地（localStorage）
- 不會傳送到任何第三方伺服器
- 您可以隨時清除瀏覽器資料來移除設定
