# 📋 活動企劃管理系統

一個功能完整的活動企劃工具，幫助您規劃和管理各類活動。

## ✨ 功能特色

### 📌 基本資訊
- 活動主題 / 名稱
- 活動日期與時間
- 活動地點
- 活動目的
- 目標對象
- 預算金額

### 📦 前置準備
- **座位安排**：支援多種座位排列方式
  - 劇院式
  - 教室式
  - U型
  - 圓桌
  - 宴會式
- **茶點準備**：管理茶點清單、數量、價格和供應商
- **待辦清單**：追蹤準備事項進度，支援優先級設定

### ⏰ 流程規劃
- 視覺化時間軸（08:00 - 20:00）
- 拖拽式流程安排
- 顏色標記區分不同流程
- 負責人指派

### 📝 活動內容
- 活動簡介
- 活動項目列表（可新增/刪除/排序）
- 所需物資設備
- 注意事項

### 👥 參與人員
- 人員角色分類（主辦、工作人員、講者、參加者）
- 搜尋與篩選功能
- 聯絡資訊管理

### 📤 匯出功能
- 一鍵匯出完整企劃書（HTML 格式）
- 可列印或轉換為 PDF

## 🚀 使用方式

### 直接開啟

```bash
# 使用瀏覽器開啟
open /Users/leegary/小程序/活動企劃/index.html
```

或直接在 Finder 中雙擊 `index.html` 檔案。

### 使用本地伺服器（可選）

```bash
cd /Users/leegary/小程序/活動企劃
python3 -m http.server 8080
# 然後在瀏覽器開啟 http://localhost:8080
```

## 📁 檔案結構

```
活動企劃/
├── index.html    # 主頁面
├── style.css     # 樣式表
├── script.js     # JavaScript 邏輯
└── README.md     # 說明文件
```

## 💾 資料儲存

- 所有資料自動儲存在瀏覽器的 LocalStorage
- 點擊「儲存」按鈕可手動儲存
- 資料僅存於本機，不會上傳到任何伺服器

- 響應式佈局，支援手機和平板
- 流暢的動畫效果
- 直覺的操作介面
